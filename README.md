# Weather API APP Project

## Description

Weather api app built with html css js nodejs and express (Running at localhost:8000)

## Prerequisites

you need these packages to run the server successfully.

1. express.
2. cors
3. pre-installed latest node-version

## Installing

1. npm install.
2. node server.js
